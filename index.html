Keyboard-driven Window Tiling for your existing X11 window manager

QuickTile is a simple attempt to duplicate the functionality of WinSplit Revolution for X11-based desktops. (eg. Linux)

You can also think of it as a standalone alternative to the keyboard-related features of the Compiz Grid plugin.

Note: QuickTile cannot be demonstrated by screenshots and I have not yet had time to record some demo videos, so please see the demos on the WinSplit site for examples of what purpose this tool serves.

Dependencies {% include apt_btn.html packages="python,python-gtk2,python-xlib,python-dbus" %}
Python 2.4+, PyGTK 2.2+, and (optional) python-xlib or dbus-python
Caveats
There are a few minor known bugs and the minimum dependency versions have not been re-confirmed recently.
Install
  • Just unpack and run. No installation necessary.
  • Keybindings for --daemonize can be edited in ~/.config/quicktile.cfg
  • To install QuickTile system-wide and add it to your desktop's list of programs to be started on login, run sudo ./setup.py install
