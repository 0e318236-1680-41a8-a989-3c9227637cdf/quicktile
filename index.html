quicktile by ssokolow

An X11 analogue to WinSplit Revolution for people who don't want to use the Compiz Grid plugin

Dependencies

Python 2.3+ PyGTK python-xlib (optional, needed for keybinding)

Install

Just unpack and run. No installation necessary.

License

GNU GPL 2.0 or later

Contact

Stephan Sokolow (deitarion/SSokolow)

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/ssokolow/quicktile

get the source code on GitHub : ssokolow/quicktile
