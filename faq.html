Frequently/Potentially Asked Questions

I have monitors of different sizes. How do I make QuickTile work properly?

This happens because the current (easy) method QuickTile uses to avoid overlapping your DE's widget panels (the _NET_WORKAREA WM hint) can only return a rectangle.

If you don't mind QuickTile positioning your windows under/over any widget panels your desktop environment provides, you can either set UseWorkarea = False in ~/.config/quicktile.cfg or run QuickTile with the --no-workarea switch to fix the problem.

If neither choice really appeals to you and you have more free time than I do (usually none), QuickTile's source code contains partial instructions for implementing a more involved but more correct alternative.

QuickTile resizes but doesn't move my windows. What do I do?

Some window managers can be set to ignore requests by applications to move your windows. It's supposed to let you assert dominance over certain kinds of badly-designed applications but, because QuickTile is also an ordinary application in the eyes of the window manager, it's also affected.

Under Metacity, you can disable this behaviour with the following command:

{% highlight bash %}gconftool-2 --set "/apps/metacity/general/disable_workarounds" false --type bool{% endhighlight %}
I set ModMask = Control Shift Mod1 but QuickTile isn't responding. Help?
This usually happens when the Compiz Grid plugin is still consuming keystrokes. Make sure you've disabled the Grid plugin in CCSM and then restart Compiz (eg. log out and back in) to make sure it's let go of the keybindings.
QuickTile doesn't meet my needs. What can I do?

Assuming you don't feel like either contributing code or filing a feature request and waiting, the Wikipedia Tiling window manager page does contain a section listing tiling utilities for X11 desktops. (Windows too, for that matter)

QuickTile does not and will not support MacOS (though, depending on how complete X11.app is, it may work) so OSX users are invited to visit StackOverflow for a list of alternatives.
