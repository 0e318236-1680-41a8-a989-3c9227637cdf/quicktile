source code 

Format a collection as a textual table.

Parameters:
  • headers (list(str)) - Header labels for the columns
  • group_by (int) - Index of the column to group results by.
  • rows (dict or iterable of iterables)
Returns: str

Attention: This uses zip() to combine things. The number of columns displayed will be defined by the narrowest of all rows.
